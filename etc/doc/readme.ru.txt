I.    Что такое FreeRapid Downloader?
=======================================

FreeRapid downloader - простое Java-приложение для скачивания с Rapidshare и других подобных файловых сервисов.

Основные возможности:
 - поддержка одновременного скачивания с нескольких сервисов
 - скачивание с использованием прокси
 - история закачек
 - слежение за буфером обмена
 - автоматическая проверка существовния файла на сервере
 - автоматическое обновление плагинов
 - распознавание простых каптч (CAPTCHA)
 - возможность автоматического завершения работы
 - много пользовательских настроек
 - работа в ОС MS Windows, Linux и MacOS
 - разнообразное оформление (много стилей)
 - простая в управлении программа!

Дополнительно:
 - Drag&Drop для URL
 - несложный API для добавления других сервисов (в виде плагинов)

 II.    Системные требования
=======================================

Рекомендуемая конфигурация
    * Windows 2000/XP/Linux (версия ядра 2.4*) или более новая ОС
    * Процессор класса Pentium 800MHz
    * Разрешение экрана не менее 1024x768 пикселей
    * 40 MB свободного места в ОЗУ
    * 10 MB свободного места на жестком диске
    * Платформа Java 2 - требуется версия не ниже 1.6 (Java SE 6 Runtime)

Для запуска приложение требует наличия Java 6.0 (http://java.sun.com/javase/downloads/index.jsp , JRE 6).

Пользователи Linux Debian могут использовать эту команду для установки Java:
   sudo apt-get install sun-java6-jre

III.   Как дапустить FreeRapid Downloader
=======================================

Установка
------------
Распакуйте файлы в любую директорию, не содержащую в пути специальных символов (например, '+' или '!').
Если вы обновляете до новой версии, можете удалить папку с предыдущей версией программы. Все настройки
пользователя сохранятся. Расположение настроек:
MS Windows: c:\Documents and Settings\_ИМЯ_ПОЛЬЗОВАТЕЛЯ_\application data\VitySoft\FRD
            и в ключ в реестре: HKEY_CURRENT_USER\Software\JavaSoft\Prefs\vitysoft\frd
Linux: ~/.FRD

НЕ ПЕРЕЗАПИСЫВАЙТЕ текущую версию более новой во избежание ошибок 


Запуск
-----------
Windows
 Просто запустите frd.exe или frd.jar

Linux
 Запустите команду ./frd.sh
 (но для начала задайте необходимые права запуска для файла)

Другие платформы
 Запустите команду java -jar frd.jar


Дополнительные команды для запуска:

java -jar frd.jar [-h -v -d -r -D<переменная>=<значение> -m -p]

параметры:
  -h (--help,-?)      показывает сообщение о поддерживаемых командах
  -v (--version)      показывает информацию о версии без запуска программы
  -d (--debug)        показывает отладочную информацию
  -r (--reset)        восстановление параметров пользователя к начальным значениям  
  -m (--minim)        свернуть главное окно при запуске  
  -Dproperty=value    изменяет значения переменных программы (используется для отладки или тестирования)
  -p (--portable)     все конфигурационные файлы будут сохранены в папке 'config', все пути к файлам будут
                      создаваться относительно папки FRD (если возможно) - полезно при запуске с USB FLASH
                      устройств

Если значение для параметра в -D будет установлено в 'default' (без '), то будет использоваться стандартное значение программы.

Пример - запуск программы в режиме отладки:
  Windows OS: frd.exe --debug
  Linux/MacOS: java -jar frd.jar --debug


Дополнительная информация:
  - Неофициальное обучения для пользователей Linux - Как настраивать FreeRapid Downloader в Linux (на испанском)
    http://manualinux.my-place.us/freerapid.html
    

IV.    Известные проблемы и их решения
=======================================
- Приложение не запускается если в пути присутствуют специальные символы типа '+' или '%'
  - X пожалуйста, переместите папку с программой в другое место не содержащее в пути подобные символы
- ESET "Smart" Antivirus в ОС Windows бокирует запуск FRD
  - X настройте ваш антивирус или запустите FRD следующим образом: frd.exe -Doneinstance=false
- Всегда корректно завершайте работу FRD, иначе потеряете список скачиваемых файлов (например, принудительное выключение Windows...)          
- Выделение "сверху вниз" элементов в главной таблице при активном процессе скачивания частично исчезает :-(
    X выделяйте строки таблицы с помощью ctrl+щелчок мыши или выделяйте элементы снизу вверх
- Модуль интерфейся вызывает org.jvnet.substance.api.UiThreadingViolationException: Component creation must be done on Event Dispatch Thread
    X игнорируйте это исключение в app.log
- исключение java.lang.UnsupportedClassVersionError
    X Вы используете старую версию Java, обновите Sun Java до 6 версии или новее
- DirectoryChooser вызывает java.lang.InternalError или виснет в Win Vista (64bit)
    X игнорируйте это исключение в app.log
- java.lang.ClassCastException: java.awt.TrayIcon cannot be cast to java.awt.Component
    X игнорируйте это исключение в app.log    
- Пользователи Linux не наблюдают иконку в трее Linux
    X есть только одно решение этой проблемы - обновление JRE до версии 1.6.0_10-rc или выше
- Ограничение: версия 0.7x более не может быть использовано если вы запустили версию 0.8 или новее (единственное решение - удалить конфигурационные файлы при возвращении на устаревшую версию)

IV.    Что делать при ошибке?
=======================================
1. Сначала прочитайте внимательно пункт IV - в нем описаны известные проблемы и ограничения
2. Не пробовали ли вы перезапустить приложение? :-)
3. Зайдите на домашнюю страничку http://wordrider.net/freerapid и/или трекер ошибок http://bugtracker.wordrider.net/
   для поиска возможного упоминания о похожей ошибке
4. Попробуйте удалить конфигурационные файлы (их расположение описано в разделе VI - Установка)  
5. Запустите приложение в режиме отладки:
   Windows OS: frd.exe --debug
   Linux/MacOS: java -jar frd.jar --debug
6. Сообщите о проблеме, используя файл app.log, как описано в разделе VI

VIII.   FAQ
=======================================

Q: Почему вы создали свой "RapidShare Downloader"?
A: 1) Потому что я не хочу зависеть от сторонних приложений, которые могут содержать много вирусов.
   2) Потому что я могу просто установить автоматическое скачивание самостоятельно. 
   3) Потому что другие программы для скачивания имеют неинтуитивный интерфейс и в них отсутствуют важные возможности.
   4) Потому что я могу.

Q: Как включить поддержку команд перезагрузки/выключения в Linux и MacOS?
A: Пожалуйста посмотрите настройки в файле 'syscmd.properties', расположенном в папке с программой.

Q: Где расположены конфигурационные файлы?
A: Смотрите раздел IV - Установка.

Q: Как использовать премиум-аккаунт Rapidshare в программе?
A: По умолчанию используется плагин Rapidshare для бесплатного досутпа к сервису.
Для использования премиум-аккаунта проследуйте Опции->Параметры...->Плагины->вкладка "Настройки", затем найдите плагин Rapidshare_premium и активируйте его, поставив галочку в столбце Х.
Для ввода своих логина и пароля нажмите на кнопку "Опции". Подтвердите ввод нажатием ОК.

Поддерживаются только аккаунты с цифровым ID. Другие возможности для премиум-аккаунтов еще не разработаны.

Q: Пользователи ОС Windows: Как запустить программу с помощью командной строки?
A: Меню Пуск->Выполнить... В появившемся окне наберите:
cmd ENTER
cd path_to_freerapid_directory' ENTER
frd.exe ENTER

если необходимо ввести параметры, используйте, например, команду frd.exe --portable
